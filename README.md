## Resume Website for Karl Haworth

### Print styles produce PDF resume. Best printed in Chrome.

#### Technologies Utilized

- [Bootstrap](https://getbootstrap.com)
- [Sass](https://sass-lang.com)
- [Eleventy](https://www.11ty.dev)


#### RUN WATCH

```bash
$ npm run start
```

#### RUN BUILD

```bash
$ npm run build
```
